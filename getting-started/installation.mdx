## 1. Install

We recommend using `create-email`, which sets up everything automatically for you.

<CodeGroup>

```sh npm
npm install svelvet
```

```sh yarn
yarn install svelvet
```

```sh pnpm
pnpm install svelvet
```

</CodeGroup>
