## Description

```HTML
<script>
	import { Controls } from '$lib';
</script>

<Controls let:zoomIn let:zoomOut let:reset let:unhideAll corner="SW">
	<button class="my-zoom-in" on:click|stopPropagation={zoomIn}/>
	<button on:click|stopPropagation={zoomOut}/>
    <button on:click|stopPropagation={zoomOut}/>
</Controls>

<style>
.my-zoom-in {
    border: none;
    background-color: black;
}
</style>
```
